# Textra Pro

Advanced messaging application for Android with intelligent call management,
privacy controls, and modern architecture.

## Overview

Textra Pro is a feature-rich SMS and MMS application built with modern Android
development practices. It focuses on three core areas:

- Intelligent call-to-message continuity
- Privacy-first architecture
- Clean, performant codebase

## Technical Architecture

The project follows Clean Architecture principles with MVI pattern for presentation
layer. It is organized into 15+ Gradle modules for separation of concerns.
